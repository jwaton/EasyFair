detect.elift(observed_y, sensitive_values, sensitive_group_indicator)¶

Calculates elift ratio for given data set. R = P(y=1|s=1)/P(y=1) Similar to impact ratio, but does not divide by general group in denominator. observed_y: numpy array, Target variable observations in the dataset

Parameters:
  • sensitive_values (pandas.Series) – Values for sensitive attribute,(need not be binary)
  • sensitive_group_indicator (int) – Indicates what is the value of the protected group. <for example in two groups (0 or 1), 1 indicates a sensitive group>
Returns:

score – measure of discrimination
Return type:

float
detect.impact_ratio(observed_y, sensitive_values, sensitive_group_indicator)¶

Calculates impact ratio(also called slift) for given data set. Ratio of positive outcomes for the protected group over the general group. R= P(y=1|s=1)/P(y=1|s=0).

Parameters:
  • observed_y (np.array) – Target variable observations (Binary).
  • sensitive_values (pd.Series) – Values for sensitive attribute
  • sensitive_group_indicator (int) – Indicates what is the value of the protected group. <for example in two groups (0 or 1), 0 indicates a sensitive group>
Returns:

score – measure of discrimination
Return type:

float
detect.mean_difference(observed_y, sensitive_values, sensitive_group_indicator)¶

Measures the difference between the means of the targets of the sensitive group and the general group. R = P(y=1|s=0) − P(y=1|s=1), If there is no difference, then there is no discrimination.

Parameters:
  • observed_y (np.array) – Target variable observations (Binary).
  • sensitive_values (pd.Series) – Values for sensitive attribute
  • sensitive_group_indicator (int) – Indicates what is the value of the protected group. <for example in two groups (0 or 1), 0 indicates a sensitive group>
Returns:

score – measure of discrimination
Return type:

float
detect.situation_testing(observed_y, sensitive_values, sensitive_feature_indicator, features, threshold, number_of_neighbors)¶

Measures which fraction of individuals in the protected group are considered to be discriminated against. Positive and negative discrimination is handled separately. Compares each individual to the opposite group and see if the decision would be different. With this, it signals direct discrimination for each individual.

Parameters:
  • observed_y (np.array) – Target variable observations (Binary).
  • sensitive_values (pd.Series) – Values for sensitive attribute
  • sensitive_group_indicator (int) – Indicates what is the value of the protected group. <for example in two groups (0 or 1), 0 indicates a sensitive group>
  • features (list[list]) – features that describe the instances.
  • threshold (float) – Threshold for tolerable difference
  • number_of_neighbors (int) – Number of neighbours
Returns:

score – measure of discrimination
Return type:

float
