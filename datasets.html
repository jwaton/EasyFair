datasets.utils module¶

datasets.utils.LoadAdultDataset(filepath)¶
Format the dataframe of adult dataset. The following transformations are made:
  1. All categorical variables are mapped to one hot vectors.
  2. The values of the income income is mapped to 1 and 0.
  3. Normalize all the numerical features.
  4. Remove rows with country name Holand-Netherlands’.
Parameters:
  • filepath (string, full path of adult dataset csv file downloaded) –
  • UCI repository. (from) –
Returns:

  • dataframe (pd.DataFrame, object containing the features.)
  • y (pd.Series, object containing binary outputs.)
